## 4.1 数组

```go
var q [3]int = [3]int{1,2,3}
r := [...]int{1,2,3}
s := [...]{99: -1}
```

- 长度是数组类型的一部分，必须是常量表达式。

- 如果数组元素类型是可比较的，那数组也是可比较的

- 在函数调用中，数组作为参数是被看成**值传递**，所以要修改数组，需传入数组的地址

```go
func zero(ptr *[32]byte) {
    for i := range prt {
        ptr[i] = 0
    }
}
//或者
func zero(prt *[32]byte) {
    *ptr = [32]byte{}
}
```
